Wat is het verschil met de script tag boven in te laden
in de head tag t.o.v onder inladen aan het einde van je HTML tag?

In head: de browser pauzeert de verwerking
van de HTML totdat de scripttaak is voltooid.

In body: de browser kan de HTML eerst volledig
parsen en de pagina aan de gebruiker tonen, voordat
de scripts worden gedownload en uitgevoerd.

Wat doet de attribuut “defer” in de script tag?

Het vertelt de browser om het script parallel
aan het parsen van de HTML-pagina te downloaden

Wat is het verschil tussen HTML en JavaScript?

Html is een opmaaktaal en JavaScript een programmertaal.

Wat is het verschil tussen innerHTML en innerText?

innerText retoneert allen de zichtbare gerendert text.

innerHTML retoneert alle inhoud binnen een element.
